# 📍 Level 2 - Task 3: Geographic Analysis

## 📖 Objective

The objective of this task is to analyze the restaurant dataset to:

- Plot restaurant locations on an interactive map using latitude and longitude coordinates.
- Identify patterns or clusters of restaurants in different geographic areas.
- Determine the cities with the highest number of restaurants.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Folium
- Visual Studio Code

---

## 📂 Project Files

- Dataset.csv
- task3.py
- Task3_Result.csv
- Restaurant_Locations_Map.html
- README.md

---

## 🚀 Steps Performed

1. Imported the dataset using Pandas.
2. Calculated the center coordinates for the map.
3. Created an interactive map using Folium.
4. Plotted restaurant locations using latitude and longitude.
5. Displayed restaurant information on map markers.
6. Counted the number of restaurants in each city.
7. Saved the top city counts to **Task3_Result.csv**.
8. Saved the interactive map as **Restaurant_Locations_Map.html**.

---

## 📊 Output

The program provides:

- Interactive map showing restaurant locations.
- Top cities with the highest number of restaurants.
- Geographic visualization to identify restaurant clusters.

The results are saved in **Task3_Result.csv**, and the map is saved as **Restaurant_Locations_Map.html**.
