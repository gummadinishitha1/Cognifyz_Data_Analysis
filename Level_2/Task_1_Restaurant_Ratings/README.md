# 📍 Level 2 - Task 1: Restaurant Ratings

## 📖 Objective

The objective of this task is to analyze the restaurant dataset to:

- Analyze the distribution of aggregate ratings.
- Determine the most common rating.
- Calculate the average number of votes received by restaurants.
- Visualize the distribution of aggregate ratings using a histogram.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Visual Studio Code

---

## 📂 Project Files

- Dataset.csv
- task1.py
- Task1_Result.csv
- Aggregate_Rating_Distribution.png
- README.md

---

## 🚀 Steps Performed

1. Imported the dataset using Pandas.
2. Analyzed the distribution of aggregate ratings.
3. Identified the most common restaurant rating.
4. Calculated the average number of votes received by restaurants.
5. Created a histogram to visualize the rating distribution.
6. Saved the analysis results to **Task1_Result.csv**.
7. Saved the histogram as **Aggregate_Rating_Distribution.png**.

---

## 📊 Output

The program provides:

- Distribution of aggregate ratings.
- Most common restaurant rating.
- Average number of votes received by restaurants.
- Histogram showing the distribution of aggregate ratings.

The results are saved in **Task1_Result.csv**, and the chart is saved as **Aggregate_Rating_Distribution.png**.
