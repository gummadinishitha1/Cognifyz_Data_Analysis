# 📍 Level 2 - Task 4: Restaurant Chains

## 📖 Objective

The objective of this task is to analyze the restaurant dataset to:

- Identify restaurant chains present in the dataset.
- Analyze the ratings and popularity of different restaurant chains.
- Visualize the most popular restaurant chains using a bar chart.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Visual Studio Code

---

## 📂 Project Files

- Dataset.csv
- task4.py
- Task4_Result.csv
- Restaurant_Chains.png
- README.md

---

## 🚀 Steps Performed

1. Imported the dataset using Pandas.
2. Identified restaurant chains based on repeated restaurant names.
3. Calculated the average rating and average votes for each restaurant chain.
4. Sorted restaurant chains by their average ratings.
5. Created a bar chart showing the top restaurant chains by number of outlets.
6. Saved the analysis results to **Task4_Result.csv**.
7. Saved the chart as **Restaurant_Chains.png**.

---

## 📊 Output

The program provides:

- List of restaurant chains in the dataset.
- Average ratings of restaurant chains.
- Average votes received by restaurant chains.
- Bar chart showing the most popular restaurant chains.

The results are saved in **Task4_Result.csv**, and the chart is saved as **Restaurant_Chains.png**.
