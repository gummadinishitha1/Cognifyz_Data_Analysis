# 📍 Level 2 - Task 2: Cuisine Combination

## 📖 Objective

The objective of this task is to analyze the restaurant dataset to:

- Identify the most common combinations of cuisines.
- Determine whether certain cuisine combinations tend to have higher restaurant ratings.
- Visualize the most common cuisine combinations using a bar chart.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Visual Studio Code

---

## 📂 Project Files

- Dataset.csv
- task2.py
- Task2_Result.csv
- Cuisine_Combination.png
- README.md

---

## 🚀 Steps Performed

1. Imported the dataset using Pandas.
2. Identified the top 10 most common cuisine combinations.
3. Calculated the average rating for each cuisine combination.
4. Identified cuisine combinations with the highest average ratings.
5. Created a bar chart to visualize the most common cuisine combinations.
6. Saved the analysis results to **Task2_Result.csv**.
7. Saved the chart as **Cuisine_Combination.png**.

---

## 📊 Output

The program provides:

- Top 10 most common cuisine combinations.
- Average ratings for cuisine combinations.
- Bar chart showing the most common cuisine combinations.

The results are saved in **Task2_Result.csv**, and the chart is saved as **Cuisine_Combination.png**.
