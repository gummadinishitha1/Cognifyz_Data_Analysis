# 🍽️ Level 1 - Task 1: Top Cuisines Analysis

## 📌 Objective
Analyze the restaurant dataset to identify the top three most common cuisines and calculate the percentage of restaurants serving each cuisine.

## 🛠️ Tools Used
- Python
- Pandas
- Visual Studio Code

## 📂 Project Files
- Dataset.csv
- task1.py
- Top_3_Cuisines_Result.csv

## 🚀 Steps Performed
1. Loaded the restaurant dataset using Pandas.
2. Removed missing values from the **Cuisines** column.
3. Split multiple cuisines into individual cuisine names.
4. Counted the frequency of each cuisine.
5. Identified the top three most common cuisines.
6. Calculated the percentage of restaurants serving each top cuisine.
7. Saved the final results to a CSV file.

## 📊 Output
The program displays:
- Top 3 most common cuisines
- Number of restaurants serving each cuisine
- Percentage of restaurants serving each cuisine

The results are also saved in **Top_3_Cuisines_Result.csv**.
