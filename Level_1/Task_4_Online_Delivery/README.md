# 📍 Level 1 - Task 4: Online Delivery Analysis

## 📖 Objective

The objective of this task is to analyze the restaurant dataset to:

- Determine the percentage of restaurants that offer online delivery.
- Compare the average ratings of restaurants with and without online delivery.
- Understand the relationship between online delivery availability and restaurant ratings.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Visual Studio Code

---

## 📂 Project Files

- Dataset.csv
- task4.py
- Task4_Result.csv
- README.md

---

## 🚀 Steps Performed

1. Imported the dataset using Pandas.
2. Counted the number of restaurants with and without online delivery.
3. Calculated the percentage of restaurants offering online delivery.
4. Computed the average restaurant ratings for restaurants with and without online delivery.
5. Displayed the analysis results in the terminal.
6. Saved the final results to **Task4_Result.csv**.

---

## 📊 Output

The program provides:

- Number of restaurants with and without online delivery.
- Percentage of restaurants offering online delivery.
- Average ratings of restaurants with online delivery.
- Average ratings of restaurants without online delivery.

The results are also saved in **Task4_Result.csv**.
