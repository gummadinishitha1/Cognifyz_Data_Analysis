# 📍 Level 1 - Task 2: City Analysis

## 📖 Objective

The objective of this task is to analyze the restaurant dataset to:

- Identify the city with the highest number of restaurants.
- Calculate the average restaurant rating for each city.
- Determine the city with the highest average rating.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Visual Studio Code

---

## 📂 Project Files

- Dataset.csv
- task2.py
- City_Analysis_Result.csv
- README.md

---

## 🚀 Steps Performed

1. Imported the dataset using Pandas.
2. Counted the number of restaurants in each city.
3. Identified the city with the highest number of restaurants.
4. Calculated the average restaurant rating for each city.
5. Identified the city with the highest average rating.
6. Saved the analysis results to a CSV file.

---

## 📊 Output

The program provides:

- City with the highest number of restaurants.
- Average restaurant rating for each city.
- City with the highest average rating.

The results are also saved in **City_Analysis_Result.csv**.
