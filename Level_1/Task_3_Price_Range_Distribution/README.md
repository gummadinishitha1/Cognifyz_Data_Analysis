# 📍 Level 1 - Task 3: Price Range Distribution

## 📖 Objective

The objective of this task is to analyze the restaurant dataset to:

- Visualize the distribution of restaurants across different price range categories.
- Calculate the percentage of restaurants in each price range.
- Understand how restaurants are distributed based on their pricing.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Visual Studio Code

---

## 📂 Project Files

- Dataset.csv
- task3.py
- Task3_Result.csv
- Price_Range_Distribution.png
- README.md

---

## 🚀 Steps Performed

1. Imported the dataset using Pandas.
2. Counted the number of restaurants in each price range.
3. Calculated the percentage of restaurants in each price range category.
4. Created a bar chart to visualize the distribution of price ranges.
5. Saved the analysis results to a CSV file.
6. Saved the bar chart as an image.

---

## 📊 Output

The program provides:

- Number of restaurants in each price range.
- Percentage distribution of restaurants across price ranges.
- Bar chart showing the distribution of restaurant price ranges.

The results are also saved in **Task3_Result.csv**, and the visualization is saved as **Price_Range_Distribution.png**.
